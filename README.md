# House Price Prediction — Linear Regression Projects

A collection of linear regression implementations built from scratch using
NumPy, as part of my learning journey through Andrew Ng's Machine Learning
Specialization (Course 1). Each project predicts house prices using
Indian housing-style data, with increasing complexity.

## Projects

[01 - Single-Feature Linear Regression](./01-single-feature-linear-regression)
Linear regression using a single feature (size in sq ft) to predict
house price. Covers cost function and gradient descent implemented
manually.

[02 - Multi-Feature Linear Regression](./02-multi-feature-linear-regression)
Extends the model to multiple features (size, bedrooms, floors, age)
using vectorized NumPy operations. Covers feature scaling (z-score
normalization) and vectorized gradient descent.

## Concepts covered across both projects

Cost function (Mean Squared Error)
Gradient descent (manual implementation, no scikit-learn)
Vectorization with NumPy
Feature scaling

## Tech stack

Python, NumPy, Matplotlib
